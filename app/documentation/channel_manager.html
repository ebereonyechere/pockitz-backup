Channel Manager

This section will help you to set Auto Like/Comment Campaigns, Auto Subscription Campaigns of your Youtube Channel automation. Apart from these you can also manage youtube channel videos, playlists, upload video to channel. This section is devided int 5 parts. These are:

  1. Auto Like/Comment Campaign
  2. Auto Subscription Campaign
  3. Video Manager
  4. Playlist Manager
  5. Upload Video

Auto Like/Comment


Click action button to create youtube auto like comment campaigns.

Click create campaign, you will see pop-up window, then enter all information need for auto like comment campaign.

Enter campaign name, select channel, select auto comment template, (if you didn't create auto like comment template then create one before you create auto like/comment campaigns.) Select Search type, enter targeted keyword comma separated if you want, enable auto like & enter daily max activity, select expiry type.

  • Search Type : Videos to like/comment will be obtained by YouTube search. You can target to search by keywords or can put channel IDs directly.
  • Targeted Keywords : System will take a keyword randomly but will make sure it is least used.
  • Targeted Channel IDs : System will take a channel ID randomly but will make sure it is least used.
  • Enable Auto Like : Do you want to hit like the videos alongside auto comment?
  • Daily Max Activity : No of like/comment activity per day. Value can range from 1 to 50. Please be careful, do not spam. Larger number of activities per day can marked as spam by YouTube.
  • Expiry Type : When the campaign will stop? After a certain number of activity or after a certain date.

Auto like/Comment Report

Auto Subscription

Click action button to create youtube auto Subscription campaigns.

Click create campaign, you will see pop-up window, then enter all information need for auto Subscription campaign.

Enter campaign name, select channel, Enter Keywords (Comma separated), enter daily max activity, chose channel Published date, select expiry type and enter the value of it, then hit create campaign button.

  • Keywords : System will take a keyword randomly but will make sure it is least used.
  • Daily Max Activity : No of subscription activity per day. Value can range from 1 to 50. Please be careful, do not spam. Larger number of activities per day can marked as spam by YouTube.
  • Expiry Type : When the campaign will stop? After a certain number of activity or after a certain date.

Video Manager

In this section, you can manage your youtube channel videos very easily. You can set auto-reply specific videos from your youtube channel also you can edit your youtube existing videos from here, and to see auto reply report, also you can set rank tracking for a specific video from youtube channel and see rank tracking report, also you are able to see individual video analytics report from here.


You are able to see full channel analytics report and create linkwheel campaign from here. Now i will show you how you can use video manager, Lets start.

You can load specific number of videos from your youtube channels through the dropdown, also you can see load more button in page bottom section.

If you want to set auto for a specific video, click three dot section, you will see a dropdown menu, click "Set auto-reply" pop-up window will appear,Enter campaign name, select auto reply template then hit create campaign.Also you edit auto reply campaign.

If you want to see auto reply campaign report click "Auto reply report" (from previous dropdown) then you will see auto reply campaign report list like this, And you can create auto reply campaign from here also click "Create campaign button". If you want to stop auto reply campaign you can do it just a single click from here, click pause button and auto reply campaign will stop automatically.

Click eye button to see report details

If you want to edit video from youtube channel click "Edit Video" (from previous dropdown) then you will see pop-up window will appear you can edit your video from here, also you can set more than one localizations click "add more localizations" and category, language, privacy status, video tags, video description and video title, then lick update video.

If you want to rank tracking for specific video click "Rank Tracking" (from previous dropdown) then you will see pop-up window will appear, enter tracking name, and keyword then hit save button.Keyword tracking process by cron job.

If you want to see rank tracking for specific video click "Rank Tracking Report" (from previous dropdown) then you will see pop-up window will appear, enter from date, to date then Search.

If you want to see full video analytics to click "Analytics" (from previous dropdown) then you will see video analytics, also you will see date range analytics enter from date and to date and hit Search.

If you want to see full youtube channel analytics click "Channel Analytics" then you will see channel analytics, also you will see date range analytics enter from date and to date and hit Search.

If you want to create link wheel campaign select each video right corner checkbox and then click option dropdown will appear click create link wheel. A pop-up window will appear if want to know how link wheel works? click "What is Video Link Wheel (VLW) and how it works?" text another pop-up window will appear. After that enter link wheel campaign name, if you have Money Video IDs then enter ids comma separated and select link wheel type then hit create link wheel. Link wheel process by cron job.

If you want to link wheel campaign report then go to link wheel menu, also you can create link wheel campaign from here, follow the same process above.

If you want to details report about link then click eye button

Upload Video

In this section, you will upload video in your youtube channel using this feature.

Click "Upload New Video" then select channel after that video uploading form will appear put all the information need for video upload to your youtube channel. Enter video title, enter video description, enter video tags (comma separated), select video cetegory, select privacy type, select time zone, select schedule time, upload your video file then hit create campaign button.Video uploading feature will process under cron job.
